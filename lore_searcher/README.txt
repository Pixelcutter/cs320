Steps to run example code in unix environment:
 - create a virtual environment in current directory with: python3 -m venv env
 - activate virtual environment with: source env/bin/activate
 - download/install dependencies with: pip3 install -r requirements.txt
 - run example with: python3 lore_builder.py
